IOS开发小结

2019-08-04

背景

  • 开发stream-client, 首先开发通知功能
  • 首先使用ios本地通知来进行消息提醒

开发总结

  • 程序会进入休眠状态，再打开时候程序状态在内存中会被锁定
    • http://www.hangge.com/blog/cache/detail_811.html
    • 可以申请在后台运行10分钟 但最多只有10分钟
    • 只有几种情况可以在后台一直运行
      • 位置服务
      • 远程通知
      • 媒体播放
  • 应该使用远程推送来进行消息提醒
    • https://blog.csdn.net/weixin_36162680/article/details/83863718
    • http://www.hangge.com/blog/cache/detail_1850.html
  • Understanding The «Unexpectedly found nil while unwrapping an Optional value» Error
    • 需要了解optional value
      • https://learnappmaking.com/found-nil-while-unwrapping-optional-value/
      • let meaningOfLife:Int? = 42
      • if let label = textLabel { ... }
  • 现在使用node-apn来进行消息推送， 按照https://help.apple.com/developer-account/#/deva05921840 来配置token
