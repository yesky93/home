Flex 弹性布局总结

Flex解决了什么问题

增强了布局的语义的表达能力, 我们不需要再用晦涩的技巧去安排布局, 如页面footer效果

<body>
    <div class="wrapper">
        <p>404 not found</p>
    </div>
    <div class="footer">
        <p>Copyright (c) 2008</p>
    </div>
</body>

如果使用Flex布局, 表达清晰

body {
  display: flex;
  flex-direction: column;
}
.wrapper {
  flex: 1;
}

如果使用负边距

.wrapper {
  height: 100%;
  margin: 0 auto -2em;
}
.footer {
  height: 2em;
}
